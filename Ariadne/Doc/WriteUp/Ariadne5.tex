\begin{abstract}
  We present the fifth version of the Ariadne program for generating
  QCD cascades according to the Colour Dipole Cascade Model.

  This version is a complete re-write of the program in C++ using the
  framework of ThePEG.

  The main new features of this version is a reformulation of the
  initial-state gluon radiation and the inclusion of CKKW-matching to
  fixed order matrix elements both for \tee-annihilation
  lepton--hadron deeply inelastic scattering and hadron--hadron
  collisions.

\end{abstract}
